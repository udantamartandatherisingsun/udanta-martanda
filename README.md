# Udant Martand - Newspaper Web Portal

A full-stack newspaper web application with a frontend, admin dashboard, and backend server.

## Project Structure

- `frontend/`: Next.js application for the main reader interface.
- `admin/`: Admin dashboard for content management.
- `backend/`: Node.js/Express server handling APIs and database logic.

## Getting Started

1. Clone the repository
2. Install dependencies in each directory: `npm install`
3. Run the development servers: `npm run dev`

## Tech Stack

- **Frontend**: Next.js, Framer Motion, GSAP, Three.js
- **Backend**: Node.js, Express, MongoDB
- **Admin**: React/Next.js
